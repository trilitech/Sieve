One-time setup: Slack OAuth app

To use the "Install via OAuth" button, paste your Slack app's Client ID and Client Secret below. Sieve stores them and never asks again.

  1. Create a Slack app at api.slack.com/apps → From scratch.
  2. Under OAuth & Permissions, add bot scopes: channels:read groups:read users:read users.profile:read channels:history groups:history chat:write.
  3. Add this Redirect URL: http://localhost:19816/oauth/callback (replace localhost with your Sieve host).
  4. Copy Client ID and Client Secret from Basic Information and paste below.

After saving, the page reloads and the Install via OAuth button appears.
